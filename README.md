# BoxCo-GameStation
This is a tool for adding not detected games to the game boxroom

# Usage
First download the tool from releases section
but if you don't trust the exe file you can always download the script itself to see what it does
find the appid of your game
open the library page of the game then click the gear icon then select properties
from the properties window go to the updates tab from the left side and you will see the appid of the game

Next open the BoxCo GameStation and paste your appid or appids 
you can seperate appids with commas or spaces

once you are done click Fetch Metadata button then it will generate the required files and edit json files located at the games appdata location

if it says couldn't fetch any data, probably you typed the appid wrong
